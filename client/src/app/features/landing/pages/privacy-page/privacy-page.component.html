Privacy notice

This student project stores account and collaboration data needed to authenticate users, manage projects, and keep work history coherent inside the platform.

What data is used

User identity data, project membership data, task activity, comments, notifications, and audit events are used to make the collaboration workflows function.

Why it is used

The platform needs this data to authenticate access, enforce project permissions, show shared work context, and maintain traceability of important actions.

Scope

This page is an in-app informational notice for the current MVP, not a final legal document for production deployment.
